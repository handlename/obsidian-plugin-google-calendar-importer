# Project Overview

## Purpose
This is an Obsidian plugin project for Google Calendar Importer. It's based on the official Obsidian sample plugin template.

Obsidian is a knowledge base that works on top of a local folder of plain text Markdown files.

## Tech Stack
- **Language**: TypeScript 4.7.4
- **Runtime**: Node.js v16+
- **Build Tool**: esbuild
- **Plugin Framework**: Obsidian Plugin API
- **Linter**: ESLint with TypeScript plugin
- **Type Checking**: TypeScript compiler

## Project Type
This is an Obsidian plugin project that extends the functionality of the Obsidian note-taking application.

## Key Dependencies
- `obsidian` - Latest Obsidian plugin API
- `esbuild` - Fast JavaScript bundler
- `typescript` - TypeScript compiler
- `@typescript-eslint/*` - TypeScript ESLint support
